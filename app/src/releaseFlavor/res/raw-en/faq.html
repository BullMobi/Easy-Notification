How can I uninstall this app?
Go to Settings > Security > Device Administrators and disable Easy Notification. You can now uninstall Easy Notification normally.

How can I dismiss the permanent notifications from active mode and the lockscreen?
The notification is necessary to ensure Easy Notification isn't killed by the system. However, to hide it safely (test notifications won't function anymore!) go to Settings > Apps > Easy Notification and disable showing notifications. NOTE: Sending test notifications will not function while this is disabled.

When will secure lockscreen (pattern/PIN) be implemented?
It is not possible to create a custom, secure lockscreen for Android. Other apps that have this feature only pretend to be secure, but in the real world there's always a way to bypass them.
